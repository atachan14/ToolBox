# Clip-Path Help

`clip-path: polygon(...)` を視覚的に作るツールです。  
キャンバス編集、点テーブル編集、履歴保存に対応しています。

## 基本の流れ

1. キャンバス上を左クリックして点を追加する
2. 点をドラッグ、または画面右のリストを編集して位置を調整する
3. 下部のコード欄をクリックして CSS をコピーする

点が 3 つ未満の状態では、`polygon(...)` は確定しません。

## キャンバス操作

- `左クリック`: 点を追加
- `点を左ドラッグ`: 点を移動
- `点を右クリック`: 点を削除
- `Ctrl + 左ドラッグ`: キャンバスをパン
- `中ドラッグ`: キャンバスをパン
- `Ctrl + ホイール`: ズーム

画面下の `Cursor` は現在位置を表示します。  
Grid が有効なときは、グリッドやスナップ結果を反映した値になります。

## サイズ・Grid・Guide

- `H / W`: 出力サイズの基準値
- `px / %`: 点座標の出力単位
- `Grid`: スナップ刻み
- `Guide`: ガイド枠の表示切り替え

## 点テーブル

右側の表では点を直接編集できます。

- `X / Y` を直接編集して位置を調整
- `#` 列をドラッグして並び順を変更
- `#` 列を右クリックして、前後への挿入や削除

点の並び順は、そのまま `polygon(...)` の頂点順になります。

## Circle Tool

下部の `ToolBox` を開くと Circle Tool を使えます。

- Circle Tool を ON にすると、キャンバス上のドラッグで円ガイドを作成
- 作成時に分割数を入力すると、その円周上にスナップ点が作られます
- Grid が有効な間は、点追加や点移動時にスナップ点も利用されます

円ガイド自体は補助用で、最終コードは polygon の点列から作られます。

## コード欄

- 下部のコード欄をクリックすると、現在の `clip-path` をコピーします
- 同時に履歴へも保存されます
- 横に長いコードはホイールで横スクロールできます

## 履歴

- `履歴に保存`: 現在の shape を履歴へ保存
- `履歴を表示`: 保存済み shape の一覧を開く
- 履歴項目をクリック: コードをコピーし、キャンバスへ復元
- 履歴項目を右クリック: 履歴から削除

履歴にはコードだけでなく、作成時のサイズ情報も含まれます。

## ショートカット

- `Ctrl + Z`: Undo
- `Ctrl + Y`: Redo

## 補足

- `Reset` は点、円ガイド、ズーム、パンをまとめて初期化します。
- タブごとに編集状態は保存されます。
