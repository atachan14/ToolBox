# Gradient Help

CSS の `background` 用グラデーションを視覚的に作るツールです。  
現在は Background と Linear Gradient を中心に編集できます。

## 基本の流れ

1. 必要なら Palette で色を用意する
2. `linear` を追加する
3. キャンバスや右側の Inspector で stop を調整する
4. 下部のコード欄をクリックして CSS をコピーする

## Toolbar

- `H / W`: プレビューサイズ
- `px / %`: stop の値の基準単位
- `Grid`: stop 移動時の刻み
- `Guide`: ガイド表示の切り替え

## Palette

左下の Palette は、編集用の色一覧です。

- スウォッチをクリック: 色を選択
- `+`: 新しい色を追加
- 色入力と Alpha 入力: 選択中スウォッチを直接編集
- スウォッチ右クリック: カラーピック / 削除
- スウォッチをドラッグ: 並び替え
- `Save`: 現在の Palette を保存
- `Load`: 保存済み Palette を読み込み
- `Reset`: Palette を初期状態に戻す

選択中の色は、Background や stop の追加時に使われます。

## レイヤー

右上のボタンでレイヤーを追加します。

- `linear`: 線形グラデーションを追加
- `radial`: 現状はプレースホルダ
- `conic`: 現状はプレースホルダ

タブはレイヤーです。  
タブの右クリックでは、表示/非表示の切り替えとレイヤー削除ができます。  
Background レイヤーは削除できません。

## Background

- Background タブでは背景色を編集します
- 色欄は直接入力できます
- 右クリックの `カラーピック` でも変更できます
- キャンバス側をクリックして、Palette の選択色を適用することもできます

## Linear Gradient

Linear レイヤーでは次を編集できます。

- `deg`: 角度
- `repeat`: repeating-linear-gradient の切り替え
- stop 一覧: 色、Alpha、位置

stop の操作方法:

- キャンバス上をクリック: stop を追加
- stop をドラッグ: 位置を移動
- stop を右クリック: カラーピック / 非表示 / 複製 / 削除
- 表の値を直接編集: 色、Alpha、位置を数値で調整
- 表の並び替えや追加も可能

## コード欄

- 下部のコード欄をクリックすると、現在の `background: ...;` をコピーします
- 同時に履歴へも保存されます
- 長いコードはホイールで横スクロールできます

非表示のレイヤーや muted stop は、出力コードから除外されます。

## 履歴

- `履歴に保存`: 現在の状態を履歴へ保存
- `履歴を表示`: 保存済み一覧を開く
- 履歴項目をクリック: コードをコピーし、状態を復元
- 履歴項目を右クリック: 履歴から削除

履歴にはレイヤー構成、Palette、サイズなどの状態も含まれます。

## ショートカット

- `Ctrl + Z`: Undo
- `Ctrl + Y`: Redo

## 補足

- レイヤーはタブドラッグで並び替えできます。
- 現在の出力順は、タブ構成をもとに `background` CSS に反映されます。
- `radial` と `conic` はまだ本実装ではありません。
